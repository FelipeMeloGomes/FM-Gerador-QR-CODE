## ⚙️ Instalação dos pacotes essenciais.

Instalar os pacotes:

```
npm install
```

Iniciar o Front-End:

```
npm run start
```

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   Node Js
-   JavaScript.
-   Libs: chalk e prompt
